# 在线学习平台

## 项目介绍

本项目是基于vue2.0 + SpringBoot构建的在线学习平台，项目采用前后端分离。前后台账号密码都为admin。

##技术栈
**前端**

vue + vue-router + axios + element-ui

**后端**

springboot + mybatis + redis + shiro + Jwt

**数据库**

MySQL


## 启动项目

```
# 安装依赖
npm install

# 项目启动
npm run serve
```
