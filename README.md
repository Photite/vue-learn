# 在线学习平台

## 项目介绍

本项目是基于vue2.0 + element-ui构建的一个在线学习平台。



## 安装依赖

```
npm install
```



## 项目启动

```
npm run serve
```



## 后端接口

### User端

```bash
## User端
# 获取全部课程
/courses

# 用户注册
/register

# 用户登录
/login

# 获取视频
/

```

### Admin端

```bash
# 管理员登陆
/adminLogin

# 查看全部用户
/users

# 添加用户
/addUser

# 删除用户
/delUser

# 查看全部课程
/getCourses

# 添加课程
/addCourse

# 删除课程
/delCourse
```
